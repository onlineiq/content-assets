## Image Size Guidelines

### Website Images
- Hero images: 1920x1080px max
- Blog featured: 1200x630px
- Service illustrations: 800x600px
- Thumbnails: 400x300px

### Academy Images
- Course covers: 1280x720px
- Lesson images: 1080x608px
- Certificate templates: 2000x1414px (A4 landscape)
- Promotional banners: 1200x628px

### Social Media
- LinkedIn: 1200x627px
- Instagram: 1080x1080px
- Instagram Stories: 1080x1920px

### Newsletter
- Headers: 600px wide
- Inline images: 800px max width

### Shared Assets
- Logos: Multiple sizes (SVG preferred)
- Icons: 64px, 32px, 24px (SVG preferred)
- Backgrounds: 1920x1080px max

## File Size Guidelines
- Website images: < 250KB
- Course images: < 400KB
- Social media: < 200KB
- Newsletter images: < 500KB
- Icons/logos (SVG): < 50KB

## File Naming Convention
format: [platform]-[category]-[descriptor]-[dimension].[extension]

Examples:
- website-hero-ai-basics-1920x1080.jpg
- academy-course-prompt-engineering-1280x720.jpg
- social-linkedin-ai-tips-1200x627.png
- shared-logo-color-500x500.svg
